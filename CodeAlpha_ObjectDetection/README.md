CodeAlpha Object Detection

Description

An Object Detection web application developed using Python, Flask, and YOLOv8. It detects and identifies objects in images with high accuracy.

Features

- Real-time object detection
- YOLOv8 model integration
- Image upload support
- Fast and accurate results

How to Run

1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python detection.py
